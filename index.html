MY WORK

MAP MY PRODUCT

Technology: HTML5, JavaScript, jQuery, JSON

This project is an HTML5 Web App Mashup. It is a product locator using the Best Buy Product, Best Buy Stores, and Google Maps APIs. Search for a product and find Best Buy stores within your chosen radius that have the product in stock.

Link: Map My Product

PARTICLE JAM

Technology: HTML5, JavaScript

Particle Jam is a Javascript Web Game. Collect colored clouds that spawn to the music to get the highest score possible. Collect multiple clouds in a row to activate the points multiplier. Miss too many clouds, and the game is over.

OpenGL ENGINE

Technology: C++, OpenGL, GLUT, GLEW, SOIL

This project is the beginnings of game engine created for my data structures and algorithms class.

Features:

  • Procedurally generated and tesselated cube, sphere, cone, and cylinder meshes
  • Calculated normals and texture coordinates
  • Sphere-Sphere collision detection
  • Sphere-AABB collision detection
  • Octree collision detection
  • FPS camera controls
  • .obj model loading

HAMSTER HOUSE

Technology: Maya, C#, Unity, Photoshop

Hamster House is a project created in 5 weeks with a team of 4. Play as a hamster and complete objectives around the house before the time runs out. I was responsible for modelling and bringing the environment into Unity, as well as some coding for interactions with the environment.

DUCKS AI

Technology: C#, Unity

This project features steering and leader following behaviors for ducks in a pond. One duck freely swims around the pond, while being contained in the water, while the others follow.
